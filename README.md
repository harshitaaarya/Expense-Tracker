# Expense Tracker (Python)

A simple command-line Expense Tracker built using Python. This project allows users to record daily expenses, view all stored expenses, and calculate the total amount spent. It is designed as a beginner-friendly project to practice Python fundamentals such as lists, dictionaries, loops, conditional statements, and user input handling.

## Features

* Add a new expense
* Store expense details (date, category, description, amount)
* View all recorded expenses
* Calculate total expenses
* Menu-driven interface
* Beginner-friendly code structure

## Technologies Used

* Python 3

## Concepts Practiced

* Lists
* Dictionaries
* Loops
* Conditional Statements
* Functions (can be added in future versions)
* User Input and Output

## How to Run

1. Clone the repository:
   git clone <repository-link>

2. Navigate to the project folder:
   cd Expense-Tracker

3. Run the program:
   python expense_tracker.py
